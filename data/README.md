## Dataset Access

This repository uses the publicly available **DreamCatcher audio dataset**.

Due to dataset licensing and distribution policies, raw data is not included
in this repository.

Please follow the official DreamCatcher dataset instructions to download
the audio data and place it under this directory.

Expected structure:

data/
├── audio/
│   ├── subject_01/
│   ├── subject_02/
│   └── ...
└── labels/
